# Changelog

## Unreleased

## 1.4.1

### 新功能

- **压缩叠词**：将连续重复 ≥ 3 次的短语/单字压成「词…词！」（如「好的」×5 →「好的…好的！」），降低 CPS；编辑器工具菜单、当前条目工具栏、QC 一键修复均可用

### 设置

- **完成后压缩叠词**：设置 → 编辑 → 批量完成后；默认开启，写回字幕
- **完成后自动按 CPS 拆句**：默认改为开启（此前默认关闭）

### 字幕编辑器

- **列表全文显示**：字幕列表文本不再两行截断，长句可完整换行查看
- **QC 叠词检测**：质量检测可标出叠词嫌疑，一键修复可选压缩叠词

### 其它

- **推荐安装**：下载 `Transub-1.4.1-win.zip` 解压后运行 `Transub.exe`，或使用 `Transub-Setup-1.4.1.exe`

## 1.4.0

### 题外话

- **更新频率**：自项目开始至今几乎保持着每天更新的节奏，但目前各项功能都已较为完善且稳定。**后续会大幅度的降低更新频率**。
- **新功能**：目前各项功能已经非常强大，尤其是字幕编辑器的功能已经强于大部分同类字幕编辑器。目前软件所呈现出来的功能已经是我所能想到的几乎所有想法了。**所以对于新功能，更多的是需要依靠网友们的想法和建议了。**
- **MAC版本：**暂无MAC版本的开发计划（有开发MAC版本的开发者可借助项目开源代码只需微做适配即可制作出MAC版本）

### 新功能

- **波形编辑器：支持放大缩小波形尺寸，通过拖动、拉拽等方式快速调整字幕时长**
- 全局术语表：在翻译模式下，自动应用全局术语表（在设置内开启，默认关闭）
- 抑幻听：通过设置，可极大程度减少幻听、杂音、错误转录BGM等情况
- 自动拆句：对于单条过长的字幕按语句自动拆分（需在设置中 开启 **自动按CPS拆句**）
- 生成繁体字幕：现在可以直接生成繁体中文字幕了（在设置的输出项内的中文字幕类型，选择 繁体中文）

### 主窗口

- **独立设置窗口**：设置改为独立 BrowserWindow；从字幕编辑器打开设置时不再顺带唤起主任务窗口

### 字幕编辑器

- **波形默认开启**：打开编辑器时默认显示波形时间轴（仍可关闭；本地已保存的开关偏好优先）
- **时间轴/波形右键菜单**：在时间轴字幕块与波形对应区间上右键，可打开与字幕列表相同的上下文菜单（分割、对齐、时长调节等）
- **删除杂音**：新增「幻觉短句」规则（完毕 / ○○○ / 过短占位等）

### 设置

- **设置分区重排**：8 个页签收为 6 个——**常规**（预设/任务/设备/托盘）、**环境**（TransWithAI + FFmpeg）、**输出**、**转录**、**编辑**（编辑器 + 批量完成后）、**更多**；各区分组与说明文案更清晰，打开设置默认进入「常规」
- **中文字幕类型**：输出页可选简体中文 / 繁体中文（默认简体）；仅「翻译」任务生效，完成后用 OpenCC 做简繁转换（不再写入 initial_prompt，避免「简体中文」等提示被复述进字幕）；独立设置窗口保存后会同步到主窗口，后处理以磁盘配置为准，避免仍按旧的「简体」转换
- **抑幻听参数**：转录页可调无语音/对数概率/压缩比阈值，以及可选的幻听静音跳过；写入 TransWithAI `generation_config`，便于 BGM/静音较多时减少胡编字幕
- **内置预设「翻译 · 抑幻听」**：日语锁定 + 更严的无语音/压缩比阈值 + 较短合并上限（8s）与智能切分 20s，适合 BGM 或静音较多的内容
- **完成后自动按 CPS 拆句 / 清理杂音幻觉**：设置 → 编辑 → 批量完成后；可选写回字幕，缓解单条过长与「完毕」类短句
- **翻译句读后空格**：翻译任务完成后，在 CPS 拆句之前自动在中文 `。？！` 后补空格（已有空白/行末不重复），便于拆句识别句界
- **术语表写入提示**：默认将全局术语写入 `initial_prompt` / `hotwords`（generation_config）
- **参数试跑对比**：设置 → 更多；对勾选视频片头用两套预设各跑一遍并对比 QC

### 平台与分发

- **发行包调整**：处于 portable 版本的运行体验较差，故后续版本仅提供 `Transub-*-win.zip` 与 `Transub-Setup-*.exe`，不再提供 portable 便携版

### 其它

- **推荐安装**：下载 `Transub-1.4.0-win.zip` 解压后运行 `Transub.exe`，或使用 `Transub-Setup-1.4.0.exe`

## 1.3.2

### 平台与分发
- **打包资源封入 asar**：HTML / JS / CSS 不再解压到 exe 旁明文目录，改为打入 `app.asar`，降低被随意篡改的风险（`_internal` 等原生依赖仍保持旁路）
- **默认分发改为 zip + 便携版**：未签名时 NSIS `Setup.exe` 易被 SmartScreen / 杀软拦截；Release 主推 `Transub-*-win.zip`（解压即用）与 `Transub-*-portable.exe`。NSIS 安装包仅作可选本地构建（`npm run build:setup`），不再作为默认发布产物

### 主窗口
- **记住窗口尺寸**：关闭后再次打开恢复上次宽高、位置与最大化状态
- **新版本检查**：更加方便的检查软件新版本，快速用上新功能和bug修复

### 设置
- **默认参数调整**：与常用配置对齐——字幕格式默认仅 SRT；合并间隔 500ms / 最大时长 10s；默认不生成 .bak；托盘进度默认关；完成后 QC 检测默认开
- **系统托盘通知开关**：运行页可开关桌面通知（任务完成 / 最小化提示），默认关闭
- **检查新版本入口**：主窗口设置按钮旁增加图标，可直接检查更新

### 字幕编辑器
- **时间轴/波形可缩放平移**：默认 5× 显示；Ctrl+滚轮或按钮放大缩小，滚轮/滑条左右滚动；倍率写入本地偏好，下次自动恢复
- **新增时间轴/波形区字幕拖动调整时长**： 调整字幕时长更方便了
- **波形加载提示**：开启波形时显示进度遮罩与状态栏提示，避免长视频抽峰时界面无反馈
- **.bak 备份位置可配置**：设置 → 字幕编辑器；可选不生成（默认）、与字幕同目录、或软件数据目录下的 `backup` 文件夹
- **静音分割更精准**：粗定位 + `atrim` 精确取窗（避免关键帧偏移）；单声道高通预处理；切分点按「靠近文本断点 + 静音时长」评分；相邻条目按静音边缘分开
- **静音分割对齐断句词**：理想切点合并空格/换行、断句词与标点；连续无空格文本若匹配断句词也可静音分割

### 修复
- **确认框屏闪**：关闭确认/取消后不再 `blur` 再抢焦，避免短暂露出下层主程序窗口
- **任务栏图标**：开发模式 (`npm start`) 会把 Transub 图标写入 `electron.exe`，避免重装 Electron 后又变回默认图标；从托盘恢复时也会重新应用窗口图标；最大化改在 show 之后，避免任务栏显示宿主 exe 图标

### 其它
- **推荐安装**：下载 `Transub-1.3.2-win.zip` 解压后运行 `Transub.exe`，或使用便携版 `Transub-1.3.2-portable.exe`（均未签名，一般不会像 NSIS Setup 那样被系统拦截）

## 1.3.1

### 主窗口
- **托盘进度与 ETA**：任务进行中更新托盘提示；进度条文案显示已用时间与预计剩余（基于片长与历史吞吐）
- **托盘相关设置**：运行页可开关「任务时更新托盘进度」「开始时最小化到托盘」
- **完成后 QC 检测**（默认关）：设置 → 字幕编辑器；任务结束后仅扫描并标出问题数，不自动修复、不自动打开编辑器
- **启动加速**：窗口 `ready-to-show`、桥接按需加载；FFmpeg / TransWithAI / GPU 检测改为后台快速检查
- **任务队列**：并行探测视频；列表增量刷新；排队完成后确认是否继续
- **设置分区**：原「高级」改为「FFmpeg」；新增「更多」（导入/导出配置、检查更新、任务历史、访问官网）
- **检查更新**：安装包将主进程共用的 `src/js/*-core` 打入 asar，修复「检查软件更新」因缺少 `subtitle-glossary-core` 报错；下载进度条显示
- **取消任务**：Windows 下 `taskkill /T` 结束 infer 进程树
- **任务完成后**：可打开输出目录、播放提示音；GPU 建议可一键应用设备

### 字幕编辑器
- **多选批量**：Ctrl/Shift 点击多选；支持批量删除、合并相邻、简繁/偏移作用于选中；`Ctrl+A` 全选可见行
- **长任务可取消**：静音批处理 / 贴边 / 重转写进度条增加取消；Esc 可中止；FFmpeg 进程可终止
- **可选波形时间轴**：工具栏开关（默认关）；开启后从关联视频抽峰绘制在时间轴下方
- **自动备份与草稿恢复**：保存时写 `.bak`；脏状态定时草稿；重新打开时若草稿更新可选择恢复
- **项目术语表**：字幕旁 `{stem}.glossary.json` sidecar；编辑器术语表弹窗可切换全局/项目，扫描与统一使用合并后的有效术语表
- **简繁短语转换**：OpenCC 短语优先 + 字符映射；可选保护术语表词条（默认开启）
- **批量简繁转换**：工具菜单 → 文本 →「简繁转换」，支持简→繁 / 繁→简，可作用于全部或当前选中条目（可撤销）
- **Ctrl+↑/↓**：文字编辑框内切换上一条 / 下一条字幕
- **快捷键**：工具栏独立按钮（设置后方）

### 修复
- **设置不持久**：打包后用户设置改为写入稳定目录（NSIS → Electron `userData`；便携版 → `PORTABLE_EXECUTABLE_DIR`），不再写到每次解压的临时 exe 旁；并自动迁移旧位置的 `transub-settings.json`（感谢QQ804218提供的bug信息）
- **FFmpeg 目录“消失”**：自定义路径若落在软件安装/`_internal` 内，更新或重装会覆盖该树；现会提示改用安装目录外的路径，且检测保存失败时不再静默忽略
- **查找替换**：全部替换成功后显示「已成功替换 N 处」，不再误报「未找到匹配项」
- **确认框失焦**：确认/取消后恢复编辑器输入焦点（避免需切窗才能继续编辑）
- **任务栏图标**：开发与打包路径正确使用 Transub `app.ico`
- **日志滚动**：转录过程中，日志自动滚动至最新条目（感谢QQ804218的建议）

### 平台与分发
- **GitHub 应用更新**：设置页「检查更新」对接 GitHub Releases
  - NSIS 安装版：若 Release 含 `latest.yml`，可应用内下载并重启安装（electron-updater）
  - 便携版 / 开发模式 / 无 `latest.yml`：通过 GitHub API 比对版本并打开下载页
