# Transub v1.0.1

## 字幕编辑

- **视频同步优化**：按字幕边界定时刷新播放位置，减少无效轮询；播放头时间显示更稳定
- **CPS 目标值**：详情面板可设置目标 CPS，超出时高亮提示
- **焦点恢复**：原生对话框、确认框关闭后自动将焦点还给编辑窗口，修复 Windows 下常见失焦问题
- **快捷键**：列表聚焦、空格切换播放/暂停等交互改进
- **硬件解码**：Windows 下优先启用 H.264 / HEVC 硬件解码，提升预览流畅度
- **CLI 直达编辑**：支持 `--edit-sub=` 与 `--edit-video=` 参数，从命令行直接打开字幕编辑窗口

## 主界面与批处理

- **耗时统计**：修复跳过/完成/失败任务的已用时长显示
- **界面精简**：移除批处理完成后的成功/跳过/失败汇总行

## 其他

- FFmpeg 探测增加视频编码、分辨率信息，便于编辑窗口适配预览
